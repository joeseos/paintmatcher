# Paint Matcher

Find equivalent paints across miniature paint ranges. Matches are computed using Delta E (CIE76) perceptual color distance.

Supports 840 paints across 13 ranges including Citadel, Vallejo, Army Painter, Reaper, Scale 75, and more.

## Tech

Built with SvelteKit, deployed on Netlify.

## Development

```bash
npm install
npm run dev
```
